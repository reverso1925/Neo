# Neo
Neo é o servidor de Minecraft mais inovador do Brasil. Survival, Skyblock, Minigames, PvP e mais.

## ✨ Funcionalidades

### 🎨 Design & Visual
| Funcionalidade | Descrição |
|---|---|
| 🌃 **Tema Cyberpunk-Neon** | Paleta escura com ciano, magenta, lime e purple neon |
| 🟫 **Estética Pixel Art** | Blocos flutuantes, ícones pixelados, fonte Press Start 2P |
| ✨ **Partículas Canvas** | Blocos de Minecraft caindo em tempo real |
| 🏙️ **Background Procedural** | Cidade cyberpunk gerada via Canvas na hero section |
| 📺 **Scan Line Overlay** | Efeito CRT sutil para reforçar a estética |
| 🔦 **Dark/Light Mode** | Toggle com ícone de tocha do Minecraft |
| 📱 **100% Responsivo** | Mobile-first com breakpoints SM/MD/LG/XL |

### ⚙️ Interatividade
| Funcionalidade | Descrição |
|---|---|
| 🔤 **Título Animado** | Letras "NEO" aparecem bloco por bloco com glow |
| 🔄 **Cards Flipáveis** | Gamemodes com hover flip (click em mobile) |
| 📊 **Contadores Animados** | Stats com contagem numérica ao scroll |
| 🛒 **Carrinho de Compras** | Sistema completo com localStorage |
| 📋 **Copiar IP** | Click-to-copy com feedback visual e toast |
| 🖼️ **Lightbox Gallery** | Modal com navegação por teclado (← → ESC) |
| 📜 **Accordion de Regras** | Expandir/recolher com animação suave |
| 🔊 **Sound Effects** | Sons pixelados via Web Audio API (hover/click) |
| 📩 **Formulário de Suporte** | Com validação, loading state e feedback |
| 🔔 **Toast Notifications** | Notificações animadas no canto da tela |
| ⬆️ **Back to Top** | Botão flutuante com aparição condicional |

### 📡 Integrações (API)
| Funcionalidade | Descrição |
|---|---|
| 👥 **Jogadores Online** | Fetch real da API `mcsrvstat.us` a cada 60s |
| 🖼️ **Skins de Jogadores** | Avatares via `mc-heads.net` API |
| 🟢 **Status do Servidor** | Indicador real-time no footer com ping |
| 💬 **Discord Widget** | Chat simulado com preview de mensagens |
| 💳 **Tebex (Placeholder)** | Estrutura pronta para integração de pagamento |

### 🔍 SEO & Performance
| Funcionalidade | Descrição |
|---|---|
| 🏷️ **Meta Tags Completas** | Title, description, keywords otimizados |
| 📱 **Open Graph** | Preview rich para Discord, Twitter, WhatsApp |
